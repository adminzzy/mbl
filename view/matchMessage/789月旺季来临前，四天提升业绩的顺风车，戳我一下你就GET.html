对于发型师来说，没有什么事能够比见到美发大师，并且聆听他们授课更能令人兴奋了。倾听老师们的建议，以及老师自身成功经验的分享，不仅对于学生，甚至对于已经在行业里有所成就的人来说都是一件非常奇妙的事情。常听到同学咨询时会问：“四周的时间学习，我会不会吸收消化不了?”其实，大师班就是用四周的时间讲解沙宣的三个技术,从认知到会判断，再到会做。很多人觉得自己笨，假如一周五天算学一遍，那学四遍对于一个吸收慢的人来说，是不是一件好事！一定要比学一周吸收的更多，理解的更深。






何谓大师班


第一周，经典课程 ( 5天 )

主讲：谢红宇

谢红宇主讲经典课程，让你在眼、脑、手三个方面对沙宣技术进入理解，知道它长什么样，有什么用，用什么方法帮助你从不会到会。


第二周，当代设计课程 ( 5天 )

主讲：谢红宇

当代设计课教会你们用经典课学到的：“三个技术，九种变化”，来测量和计算头发的长度。头发自然垂落的效果，就是客人喜欢的感觉。不仅如此，还能让你举一反三，针对不同发质、不同脸型、不同头骨、不同五官比例的客户阐述发型的设计原理，让你在没动刀之前，得到客人的信服，给你一次修剪的机会，增加业绩。


第三周，经典技术大全课程 ( 5天 )

主讲：张北平

通过当代设计课技术的使用，你会发现自己在单一技术控制不好的方面，这一周张北平校长会为大家再次提升技术的控制力，利用另一些经典发型，训练、展示帮助你再次提升技术控制能力。


第四周，潮流发型课程 ( 5天 )

主讲：林国华

从剖析潮流方向、设计理念、发型创作、结合烫染等元素；充分的让发型设计师快速的掌握当季潮流短发。 结合流行元素，如何打薄、去量，提升头发质感，增强发型通透性，??发型更飘逸。 手把手教会你如何处理纹理，烫前打薄，各种刘海修剪技巧。 课程总共有7-8款商业发型及时尚创意短发，从剪、烫、染、吹、造型等方面去针对潮流短发商业趋势，让你玩尽短发，得心应手。

三位导师，四周时间，实力打造。最后吧啦悄悄告诉你如何从大师班中汲取价值：

尽一切努力争取多得到老师面对面指导的机会。旁听大师课固然会让学生有所收获，而得到大师当面的指导定会让学生收获更多。而且大师会根据学生的自身状况提出相应的意见与建议，这对学生来说无疑是一笔宝贵的财富。


把你要问的问题提出来。由于大师班每个老师上课的时间有限，因此老师不可能会发现学生全部的问题。因此学生为从课程中获得更多的价值，应该主动将自己感兴趣的问题提出来，从而得到更有针对性的指导。


展现你的弱点，而不是回避它们，学生上大师课是为了解决工作中的问题，而不是为了单纯获得赞许。因此请展现自己的问题！
